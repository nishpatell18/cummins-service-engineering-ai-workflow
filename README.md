## What this project is
A prototype system with:
- Field technician mobile-style view
- Back-office management view
- AI-guided troubleshooting
- Human approval for risky actions
- Audit logs with timestamp and confidence
- Offline support + sync

## Project Structure
- `frontend/` – UI screens
- `backend/` – API + AI agents + logs
- `data/` – synthetic demo data
- `docs/` – architecture, safety, demo notes
- `scripts/` – setup/seed scripts

## Branch Workflow
- `main` = stable
- `dev` = team integration
- `feature/...` = individual work branches
